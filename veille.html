Angular

Angular

Angular est un Framework open source développé par Google et écrit en TypeScript. Il permet la création d'applications web.

Framework

Un Framework désigne un ensemble de composants logiciels qui sert à créer les fondations ainsi que les grandes lignes d'un logiciel.

TypeScript

TypeScript est un langage de programmation open source qui a pour but d'améliorer et de sécuriser la production de code JavaScript.

Contexte historique

AngularJS, sorti en 2009 et la première version d'Angular (communément appelé Angular 2+). AngularJS permet de faciliter le développement d'une application web de la manière suivante:

  • - Améliorer la testabilité du code
  • - Tester le code en temps réel
  • - Développement client et serveur en parallèle.

Le Framework permet également de rendre le HTML dynamique et permet finalement de développer ses propres balises et attributs HTML.

En 2016 sort la seconde version d'AngularJS, désormais nommée Angular. A partir de là, les développeurs du Framework ont sorti en moyenne une nouvelle version tout les 6 mois afin de garder le Framework à jour au maximum. De plus, chaque version est compatible avec celle qui la précède afin de faciliter la transition entre deux versions.

Au fil des années

Depuis la première version, Angular a apporté de nombreuses fonctionnalités aux développeurs afin de leurs faciliter la tâche. Le premier changement important entre AngularJS et Angular 2+ est le changement de langage de programmation. AngularJS était écrit en JavaScript, tandis que Angular est écrit en TypeScript.
Angular utilise par ailleurs une syntaxe différente qu'AngularJS, en utilisant des crochets pour la liaison des propriétés et des parenthèses pour la liaison d'évenements.
Beaucoup de fonctionnalités d'Angular ont été déplacées vers des modules. Un module permet de regrouper des composants, de définir leurs dépendances ainsi que leur visibilité.

Au cours des 2 dernières années

  • Incorporation d'un moteur/compilateur nommé "IVY":
    • Supprime le code inutilisé
    • Recompile uniquement les composants modifiés
    • Possibilité de garder d'anciennes fonctionnalités
  • Ajout d'un mode strict permettant:
    • D'améliorer la maintenabilité du code
    • D'aider à détecter des bugs à l'avance
    • D'effectuer des optimisations avancées
  • Notification permettant de savoir si l'application est irrécupérable
  • Amélioration de la mise en cache grâce à de meilleurs algorithmes
  • Meilleure compatibilité avec les plateformes web

Conclusion

Angular est un Framework basé sur des composants. Il inclut une librairie intégrée qui couvre une large variété de fonctionnalités (routage, gestion de formulaires, communication client-serveur...). Angular inclut également une suite d'outils de développement permettant des tests et des mises à jour du code. Angular est fait pour les développeurs travaillant seuls comme pour les projets d'équipes au sein d'entreprises. Le Framework a été conçu pour être mis à jour le plus facilement possible afin de minimiser les efforts.
